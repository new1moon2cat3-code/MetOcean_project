## 二、研究區域範圍定義

### 2.1 既有文獻定義（`Report/「東北角地區」定義.txt`）

> 本研究所稱「東北角海域」係指臺灣北部基隆至宜蘭頭城之沿岸及近海區域，涵蓋鼻頭角、龍洞灣至外澳海域，屬受東北季風及黑潮共同影響之高能海象區域。

**精確經緯度範圍：**
- 北緯 24.5°N – 25.2°N
- 東經 121.6°E – 122.2°E

### 2.2 水深分帶

| 分帶 | 水深範圍 | 特性 |
|------|----------|------|
| 近岸帶 | 0–50 m | 波浪破碎區，適合岸基觀測 |
| 中水深帶 | 50–200 m | 波浪能研究重點區域 |
| 外海帶 | 200 m+ | 黑潮流經區域，浮標觀測區 |

### 2.3 關鍵地理位置（由東北角地圖判讀）

```
基隆市  ─── 八斗子 ─── 鼻頭角 ─── 龍洞灣 ─── 金沙灣 ─── 三貂角 ─── 烏石港 ─── 頭城
    ↙                                  ↓                                  ↘
 基隆嶼                           龍洞岬                             外澳沙灘
    ↓
 東海／黑潮支流
```

## 三、測站建議位置

綜合地圖判讀、海洋能開發需求及既有規劃書，建議設置 **3 座主要測站** 及 **1 座輔助站**：

### 3.1 A 站：外海波浪浮標站（Offshore Wave Buoy Station）

| 項目 | 建議內容 |
|------|----------|
| **位置** | 龍洞東南方外海（約 25.03°N, 122.00°E） |
| **水深** | 約 80–120 m |
| **設備** | Datawell Directional Waverider 波浪浮標 |
| **功能** | 長期波浪觀測（波高、波向、週期、波浪頻譜） |

**設置理由：**
- 位於東北角外海、直接面對東北季風及黑潮，波浪能量最高
- 水深適中，適合波浪浮標錨繫部署
- 不致受到近岸地形及陸地干擾，波浪資料具代表性
- 距離龍洞海岸約 8–10 km，在 4G/5G 通訊範圍內

### 3.2 B 站：海流觀測站（Current / ADCP Station）

| 項目 | 建議內容 |
|------|----------|
| **位置（B1）** | 鼻頭角東北方外海（約 25.15°N, 121.95°E） |
| **位置（B2）** | 三貂角東南方外海（約 25.00°N, 122.05°E） |
| **水深** | 50–80 m |
| **設備** | Teledyne Sentinel V ADCP（海床固定式） |
| **功能** | 海流剖面監測（流速、流向、水深） |

**設置理由：**
- 黑潮支流流經東北角外海，此處最能代表區域海流特性
- 50–80 m 水深適合 ADCP 部署（Sentinel V 額定深度 200 m）
- 鼻頭角至三貂角外海為波浪能開發潛力區，須建立完整流場資料
- 海流與波浪資料可相互比對驗證

### 3.3 C 站：岸基綜合氣象站（Coastal Meteorological Station）

| 項目 | 建議內容 |
|------|----------|
| **位置（C1）** | 鼻頭角燈塔附近（約 25.13°N, 121.92°E） |
| **位置（C2）** | 龍洞灣北側岬角（約 25.11°N, 121.91°E） |
| **位置（C3）** | 烏石港堤防（約 24.87°N, 121.84°E） |
| **設備** | Vaisala AWS310 自動氣象站 + 潮位計 |
| **功能** | 氣象觀測（風速、風向、氣溫、氣壓、濕度）＋潮位監測 |

**設置理由：**
- **C1 鼻頭角**：突出海岸之岬角，風場無遮蔽，最適合風力觀測；已有燈塔基礎設施可供附掛
- **C2 龍洞灣**：相對遮蔽之灣澳，可作為近岸波浪對照站；龍洞資料浮標站（CWA）已有長期歷史資料可對比
- **C3 烏石港**：區域東界，可監測宜蘭外海波浪與潮位；漁港設施完善，便於維護

### 3.4 D 站（建議新增）：輔助潮位／水質站

| 項目 | 建議內容 |
|------|----------|
| **位置** | 基隆八斗子漁港（約 25.15°N, 121.78°E） |
| **設備** | 雷達波潮位計 + 水質感測器 |
| **功能** | 潮位基準校正、水溫鹽度監測 |

**設置理由：**
- 基隆港為北臺灣主要商港，已有長期潮位資料可進行校正
- 可補足研究區域西側之觀測空窗
- 水質監測（溫鹽深）可輔助海流資料判釋

## 七、補充建議

1. **站址驗證**：建議以 CWA 龍洞資料浮標站（24°01'53"N, 121°37'56"E）之歷史資料，驗證本站 A 站波浪觀測之代表性
2. **季節差異**：冬季東北季風期間建議增加觀測頻率（加密至 30 分鐘），以捕捉極端波浪
3. **錨繫設計**：A 站波浪浮標應採用雙點錨繫或 compliant 錨繫系統，以應對黑潮流速（0.3–1.5 m/s）
4. **ADCP 部署**：B1 站採用 Sentinel V（500 kHz，剖面最遠約 120 m），可完整涵蓋水深範圍並具備波浪量測能力
5. **資料整合**：建議將 CWA 既有龍洞站歷史資料納入，以延伸 baseline 至 1997 年
