> 課程：海氣象觀測

> 研究區域：臺灣東北角海域

## 一、測站配置總覽

本規劃依據東北角海域 Google Maps 衛星影像之海岸地形判讀，結合海洋能開發需求與海氣象觀測原理，共規劃 **6 座觀測站點**，分為外海波浪觀測、海流剖面觀測、岸基氣象與潮位觀測、資料接收與管理四大子系統。

## 三、測站配置說明

### 3.1 A 站：波浪浮標站（Wave Buoy Station）

**配置圖示位置**

```
        太平洋（東海）
            ↑
        ╔══════════╗
        ║   A 站   ║   ← Datawell Directional Waverider
        ║  ～25.03°N  ║      錨繫於水深 80–120 m
        ║  ～122.00°E ║
        ╚══════════╝
            ↑
      龍洞東南外海
            ↑
      ═══════════════
      臺 灣 陸 地
```

| 項目 | 內容 |
|------|------|
| **站碼** | A（Offshore Wave Buoy Station） |
| **位置** | 龍洞東南方外海 |
| **經緯度** | 約 25°02'N, 122°00'E |
| **水深** | 80–120 m（中水深帶） |
| **設備** | Datawell Directional Waverider Mk III |
| **觀測項目** | 顯著波高 Hs、波向、平均週期 Tz、尖峰週期 Tp、波浪頻譜、海表面溫度 |
| **取樣方式** | 每小時 10 分鐘（2 Hz），波浪加密時段可增至每 30 分鐘 |
| **傳輸方式** | 4G LTE 主線 + Iridium 衛星備援 |
| **供電方式** | 太陽能板 + 鋰電池組 |
| **維護週期** | 每 3 個月（含錨繫檢查、生箇清除、感測器校準） |

**設置理由：**

1. 位於東北角外海，正面迎向東北季風與黑潮支流，波浪能量最高，代表性最佳
2. 水深 80–120 m 適合波浪浮標標準錨繫部署（chain-wire-rope 錨繫系統）
3. 避開近岸地形折射與淺化效應對波浪資料之干擾
4. 距離龍洞海岸約 8–10 km，在 4G 通訊涵蓋範圍內
5. 可與 CWA 龍洞資料浮標站（約 25°06'N, 121°56'E）進行交叉驗證

### 3.2 B1 站：海流觀測站（ADCP Station - West）

**配置圖示位置**

```
        ╔══════════╗
        ║   B1 站  ║   ← Teledyne Sentinel V ADCP (500 kHz)
        ║  ～25.15°N  ║      海床固定式，水深 50–70 m
        ║  ～121.95°E ║
        ╚══════════╝
              ↑
        鼻頭角東北方外海
```

| 項目 | 內容 |
|------|------|
| **站碼** | B1（ADCP Station - West） |
| **位置** | 鼻頭角東北方外海 |
| **經緯度** | 約 25°09'N, 121°57'E |
| **水深** | 50–70 m |
| **設備** | Teledyne Sentinel V ADCP (500 kHz) |
| **觀測項目** | 三維海流剖面（流速 u,v,w）、流向、回波強度、水深 |
| **剖面設定** | 深度 cell 間距 1 m，共 50–70 層 |
| **取樣頻率** | 每 30 分鐘（平均 5–10 pings） |
| **部署方式** | 海床固定式（底床框架／水泥錨塊） |
| **供電方式** | 內部鋰電池（510 Wh），續航力約 12 個月 |
| **資料儲存** | 內建 16 GB micro SD 卡 + 年度回收下載 |
| **維護週期** | 每 6 個月（資料回收、電池更換、生箇清除） |

**設置理由：**

1. 鼻頭角為東北角最突出之岬角，黑潮支流在此產生明顯之流場加速效應
2. 水深 50–70 m 適合 Sentinel V 之剖面能力（最遠可達 120 m）
3. 可與 A 站波浪資料進行波流交互作用分析
4. 為東北角海域西側海流代表性測站

                         ║   B2 站  ║   ← Teledyne Workhorse II ADCP

```

| 項目 | 內容 |
|------|------|
| **站碼** | B2（ADCP Station - East） |
| **位置** | 三貂角東南方外海 |
| **經緯度** | 約 25°00'N, 122°03'E |
| **水深** | 60–80 m |
| **設備** | Teledyne Workhorse II ADCP (300 kHz) |
| **觀測項目** | 三維海流剖面（流速 u,v）、流向、回波強度、水深 |
| **剖面設定** | 深度 cell 間距 2 m，共 30–40 層 |
| **取樣頻率** | 每 30 分鐘 |
| **部署方式** | 海床固定式 |
| **供電方式** | 內部電池 |
| **維護週期** | 每 6 個月 |

**設置理由：**

1. 三貂角為臺灣極東點，黑潮主流流經外海，可監測黑潮變異性
2. 與 B1 站形成東西斷面，可掌握東北角海域之海流空間分布
3. 300 kHz 之較低頻率可在較深水域獲得較長剖面距離
4. 為東北角海域東側海流代表性測站

| 項目 | 內容 |
|------|------|
| **站碼** | C1（Bitoujiao Met Station） |
| **位置** | 鼻頭角燈塔附屬用地 |
| **經緯度** | 約 25°07'48"N, 121°55'12"E |
| **設備** | Vaisala AWS310 自動氣象站 |
| **觀測項目** | 風速、風向（10 m 標準高度）、氣溫、氣壓、相對濕度、降水量 |
| **取樣頻率** | 每 10 分鐘（風資料每 1 分鐘） |

3. 位置可同時監測來自東北方之季風與來自東方之颱風風場

    │  國立臺灣海洋大學／龍洞市民活動中心 │
    │                        │
    │  ┌─┐  ┌─┐  ┌─┐  ┌─┐  │
    │  │A │  │B1│  │B2│  │C │  ← 即時資料匯入
    │  └─┘  └─┘  └─┘  └─┘  │
    │  資料伺服器 → 資料庫 → 網頁展示  │
    └──────────────────────┘
```

| 項目 | 內容 |
|------|------|
| **站碼** | E（Data Receiving & Management Station） |
| **建議位置** | 國立臺灣海洋大學（基隆）或龍洞地區之岸基設施 |
| **設備** | 資料伺服器、NAS 儲存系統、不斷電系統、備份主機 |
| **功能** | 即時資料接收、品質管控、資料儲存、網頁展示、警戒值通報 |
| **傳輸架構** | 各測站 → 4G/衛星 → 網際網路 → E 站伺服器 |

1. 海洋大學具備海洋科學專業人力與網路基礎設施
2. 便於師生進行資料分析與維護管理
3. 可與既有海氣象資料庫（CWA、海委會）進行介接整合

### 圖 6-1：測站配置總圖（圖說範本）

> **圖 6-1 臺灣東北角海域海氣象觀測系統測站配置圖**

>
